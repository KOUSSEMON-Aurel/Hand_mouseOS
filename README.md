# Hand_mouseOS (Python Version)

A modern, fast, and beautiful AI Virtual Mouse using Python, MediaPipe, and Flet.

## Features

- **Fast Hand Tracking:** Uses Google MediaPipe for low-latency detection on CPU.
- **Smooth Mouse Control:** Filtered cursor movement for precision using PyAutoGUI.
- **Modern Interface:** Beautiful Material Design 3 dashboard built with Flet.
- **Gestures:**
  - **Move:** Index finger up.
  - **Click:** Pinch (Thumb + Index).
  - **Scroll:** (Coming soon).

## Installation

1. Install requirements:

   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:

   ```bash
   python src/main.py
   ```
